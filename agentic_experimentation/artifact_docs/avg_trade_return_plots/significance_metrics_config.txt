significance_metrics_config_XXX.csv - Schema & row-key meanings

Source:
- Generated in `sweep.run_config_sweep()` via `reporting.compute_significance_metrics(curves)`.

File pattern:
- `<agentic_output_root>/run_0/avg_trade_return_plots/significance_metrics_config_000.csv`
- `<agentic_output_root>/run_0/avg_trade_return_plots/significance_metrics_config_001.csv`
- ...

Shape:
- A "row-metric table" with a single value column:
  - Index (first column) is the row key (metric name).
  - The `significance` column holds the metric values.

Columns:
- (index): metric row key (string)
- significance: statistical test outputs for the delta series (topN_return - all_models_return)

Row keys:
- n_periods: number of evaluation periods used
- mean_delta: mean(topN_return - all_models_return)
- std_delta: stdev(topN_return - all_models_return)
- t_stat: t-statistic for mean_delta != 0 (normal approximation)
- t_p_value: p-value for the t-test (lower suggests stronger evidence delta != 0)
- sign_test_p_value: p-value for a two-sided sign test (outperformance rate != 50%, ignoring ties)
- sign_test_pos_rate: fraction of periods where delta > 0 (ignoring ties)
- bootstrap_mean_ci_low: lower bound of bootstrap CI for mean_delta
- bootstrap_mean_ci_high: upper bound of bootstrap CI for mean_delta
- bootstrap_prob_mean_gt0: bootstrap-estimated probability mean_delta > 0
