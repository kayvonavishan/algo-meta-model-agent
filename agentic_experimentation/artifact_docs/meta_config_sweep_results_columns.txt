Format:
- `column_name: description`

Notes:
- Each row in `meta_config_sweep_results.csv` is one sweep configuration / parameter set.
- Many metrics come in families:
  - `core_all_models_*` vs `core_topN_*` : absolute performance for baseline universe vs meta-selected universe.
  - `rel_*` : relative performance of `topN` vs `all_models` (deltas, capture ratios, equity ratios).
  - `stab_*` : rolling-window stability metrics (mean/min/max summaries over windows).
  - `trade_*` : trade-level metrics for `all_models`, `topN`, and `relative`.
  - `sig_*` : statistical significance metrics for the delta series.

config_id: Integer id for this sweep configuration (0-based).
vol_window: Volatility lookback window used by the meta model.
alpha_low: Lower bound for an adaptive weighting/alpha parameter.
alpha_high: Upper bound for an adaptive weighting/alpha parameter.
alpha_smooth: Smoothing factor for the adaptive alpha/weighting.
momentum_lookback: Momentum lookback window used by the meta model.
delta_weight: Weight applied to a delta/relative component in scoring.
conf_lookback: Lookback window for a confidence/consensus component.
risk_lookback: Lookback window for risk estimation.
cvar_alpha: Tail probability (alpha) used for CVaR-style risk calculations.
cvar_risk_aversion: Risk aversion strength used when penalizing tail risk.
seed: RNG seed used for any stochastic sampling components.
top_n_global: Number of global selections (top-N) made by the meta model.
include_n_top_tickers: Optional override for limiting number of tickers included (blank/None if unused).
per_ticker_cap: Cap on how many models/strategies can be selected per ticker/symbol.
per_symbol_outer_trial_cap: Cap on trials/selections per symbol in the outer selection loop.
min_ticker_score: Minimum score threshold to include a ticker/symbol (blank/None if unused).
oos_start_date: Out-of-sample start date used for evaluation.
status: Status of this config run (e.g. ok/failed).
error: Error message if status indicates failure.
selections_path: Path to the saved selections for this config (if produced).
